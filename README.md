# Rent Calculator (Python)

This project calculates how much each person needs to pay when rent, food, and electricity bills are shared equally.

## Features
- Calculate individual shares of rent, food, and electricity expenses
- Simple and interactive command-line interface
- Easy to use for roommates or flat-mates

## How to Run

```bash
python rent_calculator.py
```

The program will prompt you to enter:
- Hostel/flat rent amount
- Food expenses
- Total electricity units used
- Charge per unit
- Number of persons sharing

## Example Usage

```
Enter your hostel/flat rent = 5000
Enter the amount of food ordered = 2000
Enter the total electricity units used = 100
Enter the charge per unit = 5
Enter the number of persons living in room/flat = 2

Each person will pay =  3750.0
```
